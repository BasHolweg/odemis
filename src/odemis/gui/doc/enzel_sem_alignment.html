Overview alignment
    In this tab, it is possible to align the SEM, FIB and FLM to a single coincidence point. The alignment is split in three sections:

  1. Aligning the sample stage to the coincidence height of the FIB and SEM
  2. Using SEM beam shift to set the coincidence point of the FIB and SEM
  3. Moving the objective stage to align the FLM view to the FIB/SEM coincidence point
SEM-alignment instructions
    In this section, the coincidence point of the FIB and SEM is accurately set by setting the SEM beam shift. By using the FIB to mill a small feature in the carbon foil, an accurate coincidence point between FIB and SEM is set.

  1. Using the SEM image, navigate to an area on the grid that has exposed carbon foil that will allow fast milling of an alignment feature.
  2. Set a similar magnification of both SEM and FIB using the TFS microscope software and refresh both the FIB and SEM image in Odemis. A good starting magnification for this alignment is 5000-10000x.
  3. Using the TFS microscope software, mill a small circular alignment feature in the center of the view.
  4. Refresh the FIB view to display the alignment feature on screen.
  5. Play the SEM view.
  6. Using the beam shift control buttons, center the SEM view on the alignment feature.
  7. Continue to the FLM alignment.
