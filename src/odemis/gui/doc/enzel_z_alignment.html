Overview alignment
    In this tab, it is possible to align the SEM, FIB and FLM to a single coincidence point. The alignment is split in three sections:

  1. Aligning the sample stage to the coincidence height of the FIB and SEM
  2. Using SEM beam shift to set the coincidence point of the FIB and SEM
  3. Moving the objective stage to align the FLM view to the FIB/SEM coincidence point
Z-alignment instructions
    In this section, the coincidence plane of the SEM and FIB is determined by setting the correct sample stage Z coordinate. At the correct sample stage Z, a selected feature will line up vertically (on the same horizontal line) in both SEM and FIB image.

  1. Set a similar magnification of both SEM and FIB using the microscope software and refresh both the FIB and SEM image in Odemis. A good starting magnification for this alignment is 1000-5000x.
  2. Select a feature in the SEM image and navigate so that the feature is in the center of the SEM view.
  3. Refresh the FIB view.
  4. Using the +Z and -Z buttons, line up the same feature on the horizontal line in the FIB image.
  5. Refresh both SEM and FIB view to verify the feature is aligned.
  6. Continue to the SEM alignment
