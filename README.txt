Odemis, Open Delmic Microscope Software

= Requirements = 
Linux (tested on Ubuntu 11.10) or Windows (7)
Python (v2.7)

= Basic usage =
odmeisd is the command line interface to start and manage the Odemis backend. It
should be started first.

Run as "odemisd ...", with ... replaced by the correct arguments.
See "odemisd --help" for information.
